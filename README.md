# 家政服务网站管理系统 python699

## 项目概述

家政服务网站管理系统是基于 Python Django 框架开发的，前端使用 Vue.js 结合 ElementUI 组件库进行构建。系统后端采用 MySQL 数据库进行数据存储，旨在提供高效、稳定的管理解决方案。

## 技术栈

- 后端：Python Django
- 前端：Vue.js + ElementUI
- 数据库：MySQL

## 功能模块

系统主要包括以下功能模块：

1. 用户管理
   - 用户注册与登录
   - 用户信息管理
   - 用户权限控制
2. 服务管理
   - 家政服务项目发布
   - 服务项目编辑与删除
   - 服务项目状态跟踪
3. 订单管理
   - 订单创建与处理
   - 订单状态更新
   - 订单历史记录查询
4. 财务管理
   - 收支记录管理
   - 财务报表生成

## 系统角色

- 系统管理员：负责系统的维护与用户管理。
- 服务提供者：负责发布和更新家政服务项目。
- 客户：可以浏览服务、下单和跟踪订单状态。

## 运行环境

- Python 3.x
- Django 2.x
- Vue.js 2.x
- ElementUI 2.x
- MySQL 5.x

## 部署步骤

1. 安装 Python 环境及依赖库
2. 安装并配置 MySQL 数据库
3. 部署 Django 后端服务
4. 部署 Vue.js 前端项目
5. 配置域名及 SSL 证书

## 目录结构

```
家政服务网站管理系统
├── backend                # Django 后端代码
├── frontend               # Vue.js 前端代码
└── database               # 数据库相关文件
```

## 核心亮点

- 双端分离的开发模式，便于前后端协同开发与维护。
- 基于成熟的 Django 框架，保证了系统的稳定性和安全性。
